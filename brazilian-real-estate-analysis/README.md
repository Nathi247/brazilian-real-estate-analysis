# 🇧🇷 Brazilian Real Estate Market Analysis



## 📊 Overview

This project explores housing market data in Brazil to uncover patterns in property pricing, regional differences, and key drivers of value.



The analysis focuses on transforming raw housing data into actionable insights for decision-making in real estate.



---



## 🎯 Objectives

- Clean and preprocess raw housing data

- Perform exploratory data analysis (EDA)

- Identify pricing trends across regions

- Visualize insights using charts and graphs



---



## 🛠️ Tools & Technologies

- Python

- Pandas

- NumPy

- Matplotlib

- Seaborn

- DataCamp DataLab

## 🔍 Key Insights

- Property prices vary significantly by region

- Urban areas generally have higher housing prices

- Property size and location strongly influence value

- Data cleaning is essential for accurate analysis
